# Modol3
Akademia Spring modol 3 - REST API dla listy pojazdów

Pojazd - pola: id, mark, model, color.
API obsługuje metody webowe:
GET: allCars,
GET: getByID,
GET: getByColor,
POST car,
PUT car,
PATCH ,
DELETE: deletByID,

Przy starcie aplikacji mają dodawać się 3 pozycje.

Dla ambitnych:
 zwracanie danych w postaci XML
 obsługa Swgger UI
 HATEOAS
